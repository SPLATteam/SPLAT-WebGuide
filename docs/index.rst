Documentation de l'interface SPLAT
===================================

L'**interface SPLAT** est un classeur Excel avec un ensemble de macros actionnées via des boutons ou des raccourcis clavier. Le classeur peut être utilisé conjointement avec l'interface standard du modèle MESSAGE. Le manuel a été développé pour des modèles MESSAGE_V2 à un ou plusieurs pays.

Le classeur comporte un ensemble de feuilles de calcul de base que l'utilisateur peut ajouter ou dupliquer selon les besoins pour gérer les données d'entrée et les résultats du modèle sur lequel il travaille. Ce guide décrit les fonctionnalités de la dernière version stable de l'interface Excel SPLAT.

.. note::

    Ce guide se concentre sur l'utilisation de l'**interface Excel SPLAT** pour interagir avec le logiciel MESSAGE. Pour un guide de base sur l'utilisation directe de MESSAGE, reportez-vous à :ref:`using_message`.
